# barbaric-provisions-macos

macops quick-starts - some tools for un or lightly managed systems

My need for a snappy mac leads me to nuke and pave the OS a few times a year...

This script is what I use to hit the ground running on a fresh macOS install.

Installs brew and a list of software / make sure you look over the list, some nerdy stuff you may not require.

Installs ohmyzsh and adds a few of my essential aliases a nicley maintained zsh terminal experience enhancment 

Moves the default locations for screen shots to ~/Pictures/ScreenShots

Adds 5 blank spaces to the macOS dock and sets up my essential Finder View configurations to show more things than the apple defaults
